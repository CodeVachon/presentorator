Mocha

Testing JavaScript

Mocha is a Testing framework for JavaScript and Node. It allows us to test both the Front End and Back End to help ensure a bug free experience for our end users.

As bugs are reported on various systems, developers write a test for the bug and the expected functionality. They will then fix the code to pass the test.

Tests are never deleted. So as new systems are developed, and systems are changed, we can ensure the bug does not come back!
