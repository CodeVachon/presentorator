Introduction to Cordova / PhoneGap

  • Cordova and PhoneGap are Esentially the same Project.
    • Cordova is an Apache Open Source Project
    • PhoneGap is the Adobe Port of Cordova
  • Allows for the creation of Native Apps with HTML5, CSS3, and JavaScript.
  • CLI Driven
  • Integrate with the Ripple Project for Testing
    • Ripple is NO SUBSTITUTION for Device Testing

Web Components are Awesome (and Polymer too)

  • Web Components allow you to Define and Use custom HTML elements and have them Natively Rendered by the Browser.
  • Light DOM and Shadow DOM
  • Not Fully Supported Natively by All Browsers
    • Thats where Polymer Comes in
  • This Slideshow is using Web Components and Polymer!

Getting Started with MongoDB for CFML Developers

  • MongoDB is a Document Object Database alternative to Traditional Relationship Databases
  • Fast, Strong, Flexible, and Forgiving
  • Not a replacement for Relation Dependant Data Sets
    • No Joins
    • No Transactions
    • No SQL Crossover
  • MongoDB has Extensions for Adobe ColdFusion 10 and Railo
